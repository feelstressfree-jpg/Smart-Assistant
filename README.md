# 🎙️ Smart Assistant - Stealth Translation Protocol

**Smart Assistant** ek professional web-based tool hai jo real-time mein voice-to-voice translation karta hai. Ise khaas taur par office meetings aur international calls ke liye design kiya gaya hai jahan English bhasha ki rukawat ko khatam karna ho, wo bhi bina kisi ko pata chale (Stealth Mode).

---

## ✨ Key Features

-   **Dual-Stream Translation:** English bolne par Hindi sunai dega, aur Hindi bolne par English.
-   **Stealth UI:** Website ka interface aisa hai jaise koi official office document chal raha ho.
-   **Visibility Slider:** Screen ki opacity kam karke ise transparent banaya ja sakta hai taaki piche se koi dekh na sake.
-   **Multi-Language Support:** Hindi, English, Spanish, French aur German bhashaon ka support.
-   **Privacy Focused:** Voice data server par save nahi hota, processing real-time mein hoti hai.
-   **Voice Toggle:** Aap translation ko sunne ke bajaye sirf screen par padh bhi sakte hain (Silent Mode).

---

## 🚀 Kaise Use Karein? (Quick Start)

1.  **File Download Karein:** `index.html` file ko apne computer ya phone mein save karein.
2.  **Browser mein Open karein:** File par double-click karke use **Google Chrome** ya **Microsoft Edge** mein kholein.
3.  **Permissions Allow Karein:** Browser jab Microphone ki permission maange, toh use `Allow` karein.
4.  **Settings Set Karein:**
    * `Source Language` mein wo bhasha chunein jo samne wala bol raha hai.
    * `Output Language` mein wo bhasha chunein jismein aapko sunna hai.
5.  **Activate Karein:** `ACTIVATE SMART MODE` button par click karein aur meeting shuru karein.

---

## 📱 Mobile Par Kaise Chalayein?

* Is file ko apne phone mein transfer karein.
* Phone ke Chrome browser mein `file:///` manager ke zariye ise open karein.
* Earphones ka istemal karein taaki translation sirf aapko sunai de.

---

## 🛠️ Tech Stack (Technical Details)

* **Frontend:** HTML5, CSS3 (Modern Glassmorphism UI)
* **Speech Engine:** Web Speech API (In-built Browser Recognition)
* **Translation Engine:** MyMemory API (Free Tier)
* **Speech Synthesis:** JavaScript `SpeechSynthesis` interface

---

## 🛡️ Stealth Tips for Meetings

* **Earphone Hack:** Ek earphone meeting ke liye use karein aur dusra phone/laptop ke translation ke liye.
* **Tab Name:** Browser tab ka naam "Smart Assistant" hai jo kisi official extension jaisa lagta hai.
* **Transparency:** Agar koi aapke cabin mein aaye, toh slider se screen ko halka (faded) kar dein.
